# Next

## 当前短期目标

让 AI 对话树在 ChatGPT 和 Gemini 的长对话中更稳定、更轻、更容易排查问题。近期工作优先围绕 DOM 扫描稳定性、长对话性能、状态持久化瘦身、调试日志和导出体验展开。

## 当前优先级

1. 提高 ChatGPT 和 Gemini 页面 DOM 扫描的抗改版能力，减少页面结构变化导致的漏扫或误判。
2. 保持长对话性能可控，避免全 DOM 深扫、重复克隆节点和无限增长的匹配文本。
3. 完善本地状态持久化，继续避免保存完整回答、临时字段和敏感内容。
4. 优化跳转、高亮、拖拽重排、删除、撤销、线性排序等高频交互的稳定性。
5. 完善导出、弹窗状态恢复和脱敏调试日志，方便用户反馈 bug。

## 暂时不做

- 不接入外部 AI API。
- 不做云端同步或账号体系。
- 不引入构建流程、前端框架或远程服务。
- 不扩大站点权限，除非有明确的新站点支持任务。

- 当前扩展版本：`1.22.0`。

- 下一步默认方向：优先处理用户提出的新功能或 bug；若没有具体任务，先检查扫描稳定性、长对话性能和调试日志质量。
