# AI Conversation Tree

一个运行在 `chatgpt.com` 和 `gemini.google.com` 的浏览器扩展，用来把长对话整理成可搜索、可折叠、可重排的树状面板。

- 直接加载源码即可使用，不需要构建
- 当前为纯本地硬算法版本，不依赖任何外部 AI 接口
- 支持 ChatGPT 和 Gemini 两类对话页面

## 安装方法

1. 打开 Chrome 或 Edge，进入扩展管理页
2. 打开“开发者模式”
3. 点击“加载已解压的扩展程序”
4. 选择项目目录 `ai-conversation-tree`
5. 打开 `https://chatgpt.com/` 或 `https://gemini.google.com/` 并进入任意对话

## 适用范围

- 注入到 `https://chatgpt.com/*`
- 注入到 `https://gemini.google.com/*`
- 不依赖站点私有接口，基于当前页面 DOM 分析对话结构

## 快速预览

- 在 ChatGPT 或 Gemini 页面右侧显示对话树
- 支持搜索、折叠、拖拽和重新挂接节点
- 支持删除问题、撤销最近一次结构操作、重新排序整棵树
- 支持导出 `Markdown`、`PNG`、`SVG`、`JPG`

## 核心功能

- 自动分析问题之间的父子关系
- 支持搜索问题，并用 `上一个` / `下一个` 跳转
- 支持拖拽节点，把一个问题拖到另一个问题下面
- 支持将当前问题设为上级问题、下级问题或根问题
- 支持删除当前问题，并在当前对话里忽略它
- 支持 `上一状态`，可撤销最近一次结构操作
- 支持 `重新排序`，按当前对话重新分析整棵树
- 支持导出为 `Markdown`、`PNG`、`SVG`、`JPG`
- 支持记住同一对话上次的树状态、折叠状态和忽略状态
- 支持关闭当前对话的面板，并在设置页重新开启
- 支持在设置页复制脱敏调试日志，便于排查页面扫描、渲染和跳转问题

## 导出说明

- `Markdown`
  - `完整对话`：导出当前树里保留的问题和对应回答
  - `仅问题`：只导出问题树
- `PNG / SVG / JPG`
  - 只导出问题树
- 导出文件名会优先使用当前会话标题，并附带导出时间

## 权限说明

扩展只会注入到 `chatgpt.com` 和 `gemini.google.com`。

- `storage`：保存当前对话树状态
- `tabs`：读取当前活动标签页，用于设置页判断当前对话状态
- `scripting`：在 Edge 等浏览器需要时，为当前标签页补注入脚本
- `https://chatgpt.com/*`：在 ChatGPT 页面注入面板
- `https://gemini.google.com/*`：在 Gemini 页面注入面板

## 使用说明

- 面板默认显示在页面右侧
- 点击问题标题或圆点可跳转到对应提问
- 点击方形折叠按钮可展开或收起子树
- 面板会跟随当前页面滚动，高亮当前问题
- `删除问题` 会让当前问题从树、搜索和导出中消失
- 刷新网页后，被删除的问题仍会保持忽略
- `重新排序` 会重新拉回所有问题，再按当前对话重新分析
- `上一状态` 可回到最近一次操作前的树状态
- `关闭` 支持本次对话临时关闭，或永久关闭直到你在设置页重新开启
- 遇到 bug 时，可点击扩展图标，再点击 `复制脱敏调试日志`，把剪贴板内容发给开发者排查

## 项目结构

- `manifest.json`：MV3 配置
- `content-core.js`：内容脚本共享的轻量工具
- `content.js`：主面板逻辑、扫描、树渲染、导出、交互
- `content.css`：面板与树样式
- `hard-algorithm.js`：本地硬算法提取与关系推断
- `popup.html` / `popup.js` / `popup.css`：扩展按钮设置页

## 已知限制

- 对话树基于当前页面 DOM 构建，不调用平台私有接口
- 页面没有稳定暴露每条消息的真实时间，因此树中不显示提问时间
- 页面结构若发生较大改版，扫描选择器可能需要调整
- 图片导出基于当前树视图，受当前展开状态影响
